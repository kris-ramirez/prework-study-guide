Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements dividde and describe items on a page.
  • Semantic elements make a page accessible to all users.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • CSS box model
  • *: in this rule we have declared that ALL elements on the page will have the style properties margin and padding set as 0.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git pull origin main: pulls latest main branch
  • add -A: command to commit all files in the working branch
  • -git add -A: stage changes
  • git commit -m "": associate a message with your commit
